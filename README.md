# 河南牧业经济学院查寝项目
### 使用说明

#### 1、cd进入目录backend
```
cd .\Dormitory-examination\backend
```
#### 2、选择python环境
#### 3、安装相关库
```
pip install uvicorn
pip install fastapi
pip install sqlalchemy
pip install pydantic_settings
pip install jwt
pip install passlib
pip install python-multipart
pip install bcrypt
pip install openpyxl
```

#### 4、运行main.py
```
python main.py
```
